#### 如何永久保存私钥
1. 钱包
2. 下拉（进入钱包管理界面
1. 导出（导出的私钥并非明文，别人拿到后需要知道你的密码才能解码）
2. 打印或者手抄
3. 像存折一样放入保险柜(导出的私钥数据和解码密码一定要分开存放)


```
git reset --soft HEAD^
git push -f origin
```
